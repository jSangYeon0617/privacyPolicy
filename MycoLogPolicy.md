# 개인정보처리방침

**마이코 배치 로그 (Myco Batch Log)**

> 시행일: 2026년 6월 1일  
> 최종 수정일: 2026년 6월 1일

---

## 1. 개요

마이코 배치 로그(이하 "본 앱")는 버섯 재배 취미가 및 소규모 재배자를 위한 배치 기록 관리 앱입니다.

본 앱은 **이용자의 개인정보를 외부 서버로 전송하거나 수집하지 않습니다.**  
모든 데이터는 이용자의 기기 내부에만 저장됩니다.

본 개인정보처리방침은 본 앱이 수집하는 정보의 종류, 사용 목적, 보관 방법을 안내합니다.

---

## 2. 수집하는 정보

### 2-1. 이용자가 직접 입력하는 정보

본 앱은 이용자가 직접 입력한 아래 정보를 **기기 내부 데이터베이스(Room DB)** 에만 저장합니다.

| 항목 | 저장 목적 |
|---|---|
| 배치명, 균종, 배지 재료 | 재배 배치 기록 관리 |
| 접종일, 수확일 | D+일수 계산, 수확 예측 |
| 온도, 습도, 균사 진행률 | 환경 추이 차트 표시 |
| 수확량, 품질 평가 | BER 계산, 통계 분석 |
| 관찰 메모 | 배치별 기록 보관 |
| 재배 사진 | 사진 타임라인 표시 |

### 2-2. 자동으로 생성되는 정보

본 앱은 아래 정보를 앱 기능 제공 목적으로만 기기 내부에 생성합니다.

| 항목 | 생성 목적 |
|---|---|
| 배치 고유 ID (UUID) | 데이터 식별 |
| 기록 생성일/수정일 | 데이터 정렬 및 이력 관리 |
| 연속 기록 스트릭 | 재방문 유도 기능 |

### 2-3. 수집하지 않는 정보

본 앱은 아래 정보를 **수집하지 않습니다.**

- 이름, 이메일, 전화번호 등 식별 가능한 개인정보
- 위치 정보 (GPS)
- 기기 고유 식별자 (IMEI, Android ID 등)
- 주소록, 통화 기록
- 외부 계정 정보 (Google, Facebook 등)

---

## 3. 정보의 저장 및 관리

### 3-1. 저장 위치

모든 데이터는 이용자 기기의 **앱 전용 내부 저장소**에만 보관됩니다.

```
저장 위치: 기기 내부 저장소 (Room Database)
외부 전송: 없음
클라우드 동기화: 없음
```

### 3-2. 보관 기간

- 이용자가 직접 삭제하거나 앱을 삭제하기 전까지 보관됩니다.
- 앱 삭제 시 모든 데이터가 기기에서 완전히 삭제됩니다.

### 3-3. 보안

- 기기의 기본 보안(화면 잠금, 암호화)에 준하여 보호됩니다.
- 본 앱은 별도의 서버를 운영하지 않으므로 외부 해킹 및 데이터 유출 위험이 없습니다.

---

## 4. 카메라 및 저장소 권한

본 앱은 아래 권한을 요청할 수 있습니다.

| 권한 | 요청 목적 | 거부 시 영향 |
|---|---|---|
| `CAMERA` | 관찰 및 수확 사진 촬영 | 카메라 촬영 불가 (갤러리 선택은 가능) |
| `READ_MEDIA_IMAGES` | 갤러리에서 사진 선택 | 갤러리 사진 첨부 불가 |
| `POST_NOTIFICATIONS` | 관찰 예정 및 수확 예측 알림 발송 | 알림 수신 불가 |
| `SCHEDULE_EXACT_ALARM` | 정확한 시간에 알림 발송 | 알림 시간 부정확해질 수 있음 |
| `RECEIVE_BOOT_COMPLETED` | 기기 재부팅 후 알림 재등록 | 재부팅 시 알림 초기화 |

> 모든 권한은 해당 기능 최초 사용 시에만 요청되며,  
> 기기 설정 > 앱 > 마이코 배치 로그 > 권한에서 언제든지 변경할 수 있습니다.

## 5. 광고

본 앱은 Google AdMob을 통해 광고를 표시할 수 있습니다.

### 5-1. 광고 형식

- **배너 광고**: 히스토리 화면 하단 표시
- **전면 광고**: 배치 완료 리포트 열람 전 1회 표시
- **보상형 광고**: CSV/PDF 내보내기 기능 이용 시 자발적 시청

### 5-2. AdMob의 정보 수집

Google AdMob은 광고 최적화를 위해 아래 정보를 수집할 수 있습니다.

- 광고 식별자 (Android Advertising ID)
- 기기 정보 (OS 버전, 기기 모델)
- 앱 사용 이벤트 (광고 노출, 클릭)
- 대략적인 위치 정보 (IP 기반)

> AdMob의 개인정보 처리에 대한 자세한 내용은 아래를 참고하세요.  
> [Google 개인정보처리방침](https://policies.google.com/privacy)

## 6. 제3자 제공

본 앱은 이용자의 재배 데이터를 **어떠한 제3자에게도 제공, 판매, 공유하지 않습니다.**

단, 아래의 경우에만 정보가 제3자 시스템을 거칩니다.

| 상황 | 대상 | 목적 |
|---|---|---|
| 광고 표시 | Google AdMob | 광고 식별자 기반 광고 최적화 |
| CSV 파일 공유 | 이용자가 선택한 앱 (파일 앱, 이메일 등) | 이용자 요청에 의한 파일 공유 |

> CSV 공유는 이용자가 직접 "공유" 버튼을 탭할 때만 실행되며,  
> 공유 대상 앱은 이용자가 직접 선택합니다.
